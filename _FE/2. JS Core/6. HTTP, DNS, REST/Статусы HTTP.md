- 1XX - Информация
	- 100 - Continue
- 2XX - Успешное выполнение
	- 200 - OK
	- 201 - Created
	- 202 - Accepted
	- 204 - No content 
- 3XX - Перенаправление
	- 301 - Постоянное перемещение
	- 302 - 
	- 307 - временное перенаправление
- 4XX - Ошибка в выполнении
	- 400 - Bad request
	- 401 - Unauthorized
	- 403 - Доступ запрещен
	- 404 - страница не найдена
- 5XX - Ошибка сервера
	- 500 - внутренняя ошибка сервера
	- 501 - Not Implemented
	- 502 - Bad Gateway
	- 503 - Service Unavailable
	- 504 - Gateway Timeout
